Configuration
#############
podfetchdbus looks for configuration files at
- ``/etc/podfetchdbus.conf``
- ``~/.config/podfetchdbus/podfetchdbus.conf``

The configuration looks like this:

.. code:: ini

    [podfetchdbus]
    name = value
